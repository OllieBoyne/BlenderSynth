blendersynth.annotations.keypoints

blendersynth.annotations.keypoints.project_keypoints(points3d, camera=None, scene=None)[source]

Project 3D world points to 2D image coordinates for each camera.

Parameters:

  • points3d (ndarray) – Nx3 array of 3D points

  • camera (Union[Camera, List[Camera]]) – Either a single Camera object, or a list of them. If None, use bpy.context.scene.camera

  • scene (Scene) – bpy.types.Scene (if None, use bpy.context.scene)

Return type:

AnnotationHandler
